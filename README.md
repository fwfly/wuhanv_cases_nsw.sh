# wuhanv_cases_nsw.sh

Usage
```
  wuhanv_cases_nsw.sh <postcode> <date>
  Ex: 
    - wuhanv_cases_nsw.sh 2000        # Check today's new cases
    - wuhanv_cases_nsw.sh 2000 04-08  # Check new cases on April 8th
```

Result
```
New NSW cases:       16 # ALL new cases in NSW
New 2000 Cases :     0  # New cases in post code 2000 area
Cases in 2000:       16 # ALL cases in post code 2000 area
```
